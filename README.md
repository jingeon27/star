| 메소드 | 문법 | 설명 |
| ------ | ---------------------- | --------------------------- |
| appendChild | `Node.appendChild()` | Node의 자식요소 생성 자식요소의 태그는 괄호 안으로 해당함 |
| removeChild | `Node.removeChild()` | Node의 자식요소 삭제 괄호 안에 옵션을 정할 수 있음|
